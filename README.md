# 📚 Booklog

**개발자로서 읽은 책을 기록하는 공간**

책을 읽어도 지식이 금방 휘발되는 이유는 기록의 부재 때문이라고 생각했다.
어떤 고민을 했는지, 어떻게 해결했는지, 실무에서 구체적으로 무엇이 나아졌는지를 남기기 위해 이 프로젝트를 시작했다.
기록을 데이터화함으로써 내가 얼마나 성장했는지, 또 어떤 분야가 부족한지를 스스로 점검하기 위한 나만의 이정표이다.
